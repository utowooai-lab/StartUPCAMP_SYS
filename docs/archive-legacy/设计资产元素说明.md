# 设计元素提炼与切图清单

本目录从 `/Users/woo/Downloads/营地系统设计稿` 和生成稿中提炼出可用于用户端前端实现的设计元素。

资产目录：

- `crops/`：已裁切的可复用 UI 模块和视觉元素。
- `reference-screens/`：完整页面参考图。
- `asset-contact-sheet.png`：所有裁切资产总览。

## 1. 视觉基因

### 品牌蓝顶栏

可用资产：

- `crops/top-blue-header.png`
- `crops/brand-logo-header-white.png`
- `crops/brand-logo-login-blue.png`

设计特征：

- 顶部通栏使用高饱和品牌蓝。
- Logo 在深蓝场景用白色版本，在浅色登录/注册页用蓝色版本。
- 右侧设置入口使用白色线性图标 + 文案。

前端落地：

- 顶栏高度桌面端建议 64-70px。
- 品牌区固定在左侧，设置/用户信息放右侧。
- 顶栏下方内容区使用浅蓝渐变背景，不直接接纯白。

## 2. 背景系统

可用资产：

- `crops/app-soft-gradient-background.png`
- `crops/login-left-hero-panel.png`

设计特征：

- 主应用背景是极浅蓝灰底，叠加柔和蓝色、粉色、紫色光晕。
- 登录/注册左侧使用完整蓝色插画面板，包含奔跑孩子、城市、云、价值主张。

前端落地：

- 应用主背景可用 CSS 径向渐变复刻，不建议整页铺位图。
- 登录/注册左侧插画可以作为位图资产使用。
- 工作台页面避免纯灰背景，保持轻盈的蓝色空气感。

## 3. 角色身份卡片

可用资产：

- `crops/profile-card-student-leader.png`
- `crops/profile-card-parent.png`
- `crops/profile-card-teacher.png`

设计特征：

- 浅蓝卡片底。
- 左侧大圆头像，白色描边和轻阴影。
- 右侧姓名 + 角色徽章。
- 背景右下角有半透明圆弧装饰。

可复用元素：

- 默认头像蓝色圆底。
- 角色徽章：队长、队员、家长、老师。
- 勋章小图标用于强化角色身份。

## 4. 侧栏信息卡片

可用资产：

- `crops/sidebar-my-courses-card.png`
- `crops/sidebar-material-card.png`

设计特征：

- 白色卡片，圆角 12-14px。
- 标题左侧为蓝色小图标，局部黄色点缀。
- 列表使用小蓝菱形 bullet。
- 右侧操作按钮为蓝色描边小按钮。

前端落地：

- 适合「我的课程」「相关资料」「今日资料」「待处理」等列表。
- 列表行高保持 42-46px。
- 卡片标题区和列表区之间用浅灰分割线。

## 5. 功能入口卡片

可用资产：

- `crops/feature-card-submit-idea-banner.png`
- `crops/feature-card-camp-album.png`
- `crops/feature-card-feedback.png`

设计特征：

- 浅蓝底，大号品牌蓝标题。
- 配合 3D/轻拟物图标。
- 底部或右侧有半透明圆形装饰。
- 提交创业想法卡是横向主行动，右侧箭头明确引导。

前端落地：

- 用作学生端和家长端的核心入口。
- 标题字号建议 22-28px，字重 700。
- 描述字号 14-16px，颜色使用中灰。
- hover 可轻微上浮，阴影加强。

## 6. 中央业务面板

可用资产：

- `crops/center-workshop-task-panel.png`
- `crops/parent-review-main-panel.png`
- `crops/teacher-student-table-panel.png`
- `crops/teacher-team-management-panel.png`
- `crops/generated-teacher-record-dashboard.png`
- `crops/generated-parent-report-cards.png`

设计特征：

- 主要内容区域为大白卡片。
- 圆角约 12-14px。

- 老师端允许更高密度表格，但仍保留大留白和蓝色主按钮。

可复用组件：

- 数据表：浅灰表头、状态标签、蓝色操作按钮。
- Tab：蓝色文字 + 2px 下划线选中态。
- 概览卡：图标圆底 + 数字指标 + 状态说明。

## 7. 团队展示与管理

可用资产：

- `crops/team-card-student-leader.png`
- `crops/modal-edit-team.png`

设计特征：

- 团队卡片使用右侧栏独立白卡。

## 8. 弹窗体系

可用资产：

- `crops/modal-submit-idea-form.png`
- `crops/modal-feedback-form.png`
- `crops/modal-edit-team.png`
- `crops/modal-task-submit.png`
- `crops/modal-album-grid.png`

设计特征：

## 9. 表单与上传

- `crops/modal-submit-idea-form.png`
- `crops/modal-feedback-form.png`
- `crops/modal-task-submit.png`

## 13. 状态标签与评分元素

从参考图可提炼以下状态色：

- 已完成/活跃：绿色
- 进行中/当前：品牌蓝
- 待修改/请假中：黄色
- 风险/未报到/移除：红色
- 团队名：蓝、紫、绿、红等浅色标签
- 星级评分：黄色星星 + 灰色未选中星星

适用组件：

- 学生列表状态
- 老师点评
- 任务提交
- 团队进度
- 营地报告

## 15. Figma 使用建议

当前已经从位图设计稿中切出了可复用参考资产。如果要继续使用 Figma，建议下一步不是直接把这些位图当最终组件，而是：

1. 将 `asset-contact-sheet.png` 和 `crops/` 导入 Figma 作为参考素材页。
2. 按本清单重建矢量组件、颜色变量、文字样式。
3. 用完整页面参考图校准间距、圆角、阴影和组件状态。

最适合进入 Figma 组件库的元素：

- 顶栏与 Logo 区
- 角色身份卡片
- 侧栏列表卡片
- 功能入口卡片
- 弹窗标题与表单弹窗
- 团队成员头像/徽章
- 状态标签与评分组件
