# 少年马斯克创业营营地管理系统 (Camp System)

> 基于 Vue3 + NestJS + PostgreSQL 的前后端分离多角色营地管理 SaaS 平台

## 项目概述

少年马斯克创业营营地管理系统覆盖营前报名建档、营中课程任务与老师点评、营后报告导出与复盘的全生命周期。系统分为后台管理（管理员使用）和用户端（老师/学员/家长使用）两部分。

## 技术栈

| 层 | 技术 |
| --- | --- |
| 前端 | Vue 3 + TypeScript + Vite + TDesign Vue Next + Pinia |
| 后端 | NestJS + TypeScript + Prisma ORM |
| 数据库 | PostgreSQL |
| 缓存 | Redis |
| 认证 | JWT + Refresh Token + RBAC |
| Monorepo | pnpm workspace |

## 项目结构

```
StartUpCampSaaS_Project_src/
├── packages/
│   ├── shared/     # 共享类型、枚举、Zod 校验、常量
│   ├── server/     # NestJS 后端 API
│   ├── admin/      # 后台管理前端（TDesign）
│   └── user/       # 用户端前端（老师/学员/家长）
├── docs/           # 需求文档
├── pnpm-workspace.yaml
└── docker-compose.yml
```

## 快速开始

### 环境要求

- Node.js >= 18
- pnpm >= 8
- PostgreSQL >= 14
- Redis >= 6

### 安装依赖

```bash
pnpm install
```

### 数据库配置

1. 复制环境变量文件：
```bash
cp packages/server/.env.example packages/server/.env
```

2. 创建数据库并执行迁移：
```bash
pnpm db:push
pnpm db:seed
```

### 启动开发服务

```bash
# 同时启动所有服务
pnpm dev

# 或分别启动
pnpm dev:server   # 后端 http://localhost:3000
pnpm dev:admin    # 后台 http://localhost:5173
pnpm dev:user     # 用户端 http://localhost:5174
```

## 核心功能

### V1.0 基线
- 营地/班级/队伍/营员管理
- 用户管理（学员/家长/老师）
- 课程/工作坊/任务/资料管理
- 营地相册
- 家长反馈
- 创业想法

### V1.1 增强
- 营地五态状态机（未开营/招生中/已招满/开营中/已结营）
- 全局营地上下文
- 老师点评系统（营地记录 + 任务点评 + 维度评分）
- 点评内容三层（原始记录/内部点评/对外点评）
- AI 点评美化（提示词模板 + 受控 AI 接口）
- 招生信息收集（动态表单 + 版本管理 + 审核流程）
- 内容包与发布节奏
- 运营看板
- 批量导入导出
- 老师与投资人评委邀请
- 小鹅通课程同步与授权

## 权限体系

| 角色 | 数据范围 |
| --- | --- |
| 学员 | 自己所属营地及自己的内容 |
| 家长 | 已绑定学员 |
| 带队老师 | 负责队伍学员 |
| 生活老师 | 当前营地全部基础安全信息 |
| 驻场导师 | 所带班级学员 |
| 管理员 | 授权营地全部信息 |
