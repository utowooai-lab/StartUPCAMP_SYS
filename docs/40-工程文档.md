# 工程文档

更新时间：2026-07-02  
适用目录：`StartUpCampSaaS_Project_src`

## 1. 工程定位

`StartUpCampSaaS_Project_src` 是 SaaS 主工程源码，包含后台管理端、用户端、后端 API 和共享类型。

## 2. 技术栈

| 层 | 技术 |
| --- | --- |
| Monorepo | pnpm workspace |
| 后台前端 | Vue 3 + TypeScript + Vite + TDesign Vue Next + Pinia |
| 用户端前端 | Vue 3 + TypeScript + Vite + TDesign Vue Next + Pinia |
| 后端 | NestJS + TypeScript + Prisma ORM |
| 数据库 | PostgreSQL |
| 缓存 | Redis |
| 认证 | JWT + Refresh Token + RBAC |

## 3. ID / UUID 生成规则

Prisma schema 中业务实体主键统一采用：

```prisma
id String @id @default(uuid())
```

生产数据由数据库/Prisma 生成 UUID。`packages/shared/src/utils/index.ts` 中的简化 UUID 工具主要用于前端临时对象或非数据库主键场景。`packages/server/prisma/seed.ts` 中的固定 id 是测试/初始化数据，不代表生产 ID 规则。

## 4. 后端 API 模块

| 模块 | 路径 | 主要能力 |
| --- | --- | --- |
| Auth | `/auth` | 登录、刷新 token、profile、改密、登出 |
| Users | `/users` | 用户列表、详情、新增、编辑、禁用、重置密码、日志 |
| Camps | `/camps` | 营地列表、详情、统计、配置检查、状态日志、状态变更 |
| Camp Members | `/camp-members` | 营地成员、批量加入、班级/队伍关系 |
| Teams | `/teams` | 队伍 CRUD、批量建队、状态、成员、队长 |
| Tasks | `/tasks` | 任务 CRUD、任务提交 |
| Evaluation | `/evaluation` | 营地记录、任务点评、提交审核、发布、撤回 |
| AI Beautify | `/ai-beautify` | AI 润色、采纳、放弃、调用历史 |
| Enrollment | `/enrollment` | 表单、表单版本、报名提交、审核 |
| Content Package | `/content-package` | 内容包、发布、撤回 |
| Albums / Feedback / Ideas | `/albums` `/feedback` `/ideas` | 相册、反馈、创业想法 |
| Import Export | `/import-export` | 模板、导入、导出、任务查询 |
| Files | `/files` | 文件上传 |

## 5. 共享包规则

`packages/shared` 应承载枚举、类型、Zod 校验和常量，例如角色、状态、营地状态、任务状态、评分维度、六维成长能力评估。

## 6. 数据权限原则

- JWT 用于识别当前用户。
- RBAC 控制角色能力。
- 营地上下文控制数据范围。
- 老师权限必须结合 `TeacherCampAuth` 或营地成员关系。
- 家长权限必须结合 `ParentStudentBinding`。

pnpm db:seed
pnpm dev
