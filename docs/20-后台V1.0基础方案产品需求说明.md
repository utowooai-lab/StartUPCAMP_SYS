# V1.0【营地系统后台】基础方案产品需求说明

更新时间：2026-07-02  
适用端：管理后台  
版本定位：后台基础管理能力

## 1. 版本目标

V1.0 后台目标是支撑营地基础运营：配置营地、管理用户、组织班级队伍、管理课程/工作坊/任务/资料、查看记录报告、处理相册反馈、完成导入导出。

## 2. 信息架构

| 页面 | 代码路径 | 说明 |
| --- | --- | --- |
| 工作台 | `packages/admin/src/views/dashboard` | 运营概览和关键指标 |
| 营地管理 | `packages/admin/src/views/camp` | 营地基础信息、状态、配置 |
| 用户管理 | `packages/admin/src/views/user` | 学员、家长、老师、管理员账号 |
| 营地成员 | `packages/admin/src/views/camp-member` | 用户在营地内的班级、队伍、角色关系 |
| 队伍管理 | `packages/admin/src/views/team` | 队伍、成员、队长、项目 |
| 课程/工作坊/任务/资料 | `course` / `workshop` / `task` / `material` | 内容与学习管理 |
| 记录点评 | `packages/admin/src/views/evaluation` | 营地记录、任务点评、发布 |
| 相册/反馈/Idea | `album` / `feedback` / `idea` | 运营内容和家长反馈 |
| 导入导出 | `packages/admin/src/views/import-export` | 模板、导入、导出、任务记录 |

## 4. API 对应

主要依赖 `/auth`、`/users`、`/camps`、`/classes`、`/camp-members`、`/teams`、`/courses`、`/workshops`、`/materials`、`/tasks`、`/evaluation`、`/albums`、`/feedback`、`/ideas`、`/import-export`、`/operation-logs`。

## 5. 验收标准

- 管理员可以完成一个营地从创建到成员、内容、任务、记录和反馈的基础配置。
- 后台数据可以支撑用户端展示。
- 关键操作有权限控制和错误提示。
- 导入导出任务可追踪。
