# V1.0【营地系统用户端】基础方案产品需求说明

更新时间：2026-07-02  
适用端：学员端、家长端、老师端  
版本定位：基础业务闭环

## 1. 版本目标

V1.0 用户端目标是让学员、家长和老师完成营地期间最小可用闭环：登录注册、信息建档、任务提交、课程资料查看、老师记录点评、家长查看反馈。

## 2. 用户端范围

| 角色 | 页面/模块 | 说明 |
| --- | --- | --- |
| 通用 | 登录、注册、身份选择、账号设置 | 支持学员、家长、老师进入不同流程 |
| 学员 | 首页、任务、课程、团队、创业想法、相册 | 完成学习、提交、协作与成长查看 |
| 家长 | 首页、课程、相册、点评/报告、反馈 | 查看孩子营地动态并发起反馈 |
| 老师 | 首页、学生、记录、点评 | 查看负责范围、记录观察、点评任务 |

## 3. 登录注册与身份建档

- 通用登录：用户使用手机号/账号和密码登录，登录后按角色进入对应端。
- 学员注册：身份确认、基础档案、报名营地信息、营前问卷、账号确认。
- 家长注册：了解学员问卷、安全自查、家长账号、绑定学生、确认报告。
- 老师注册：邀请注册、基础档案、营地职责、账号审核。

对应原型：`20-29-student/parent-register*.html`、`03-05-teacher-register-step*.html`。

## 4. 学员端基础功能

| 模块 | 功能要求 |
| --- | --- |
| 首页 | 展示当前营地、今日任务、课程资料、团队信息、报告入口 |
| 课程资料 | 查看课程、工作坊和资料列表，按营地/日程展示 |
| 任务提交 | 支持文本、图片、文件提交，查看提交状态和老师点评 |
| 团队管理 | 查看队伍、队长、成员、项目和团队任务 |
| 创业想法 | 提交问题、目标用户、解决方案、进展，可迭代版本 |
| 相册报告 | 查看已发布相册、成长报告和老师点评 |

## 5. 家长端基础功能

| 模块 | 功能要求 |
| --- | --- |
| 首页 | 支持切换绑定孩子，展示孩子当前营地、每日记录、课程、相册、报告和反馈入口 |
| 课程相册 | 查看孩子授权范围内课程说明和已发布相册 |
| 点评报告 | 查看已审核发布的老师点评、任务反馈和成长报告 |
| 问题反馈 | 提交问题类型、标题、描述、图片、联系方式，查看处理状态和回复 |

## 6. 老师端基础功能

| 模块 | 功能要求 |
| --- | --- |
| 首页 | 展示负责营地、班级、队伍和待处理任务 |
| 学生列表 | 按营地、班级、队伍筛选学生，查看任务/记录状态 |
| 营地记录 | 填写学生、队伍、项目或特殊事件记录，默认内部可见 |
| 任务点评 | 查看任务提交，填写点评、维度评分和对外反馈 |

## 7. 权限与可见性

| 数据 | 学员 | 家长 | 老师 |
| --- | --- | --- | --- |
| 自己任务 | 可见/可提交 | 绑定后可查看结果 | 授权范围内可点评 |
| 原始记录 | 不可见 | 不可见 | 负责范围内可见 |
| 对外点评 | 发布后可见 | 发布后可见 | 可创建/发布 |
| 相册 | 发布后可见 | 发布后可见 | 可上传/管理，视权限而定 |
| 反馈 | 不适用 | 本人可见 | 授权处理可见 |
